# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'XWikiAutomation'
Import-Module -Name 'XWikiAutomation'
```

To install privately:

```powershell
Save-Module -Name 'XWikiAutomation' -Path '.'
Import-Module -Name '.\XWikiAutomation'
```
